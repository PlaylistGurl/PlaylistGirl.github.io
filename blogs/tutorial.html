How to Upload Your Blog Post to GitHub Pages

  1. Copy your downloaded .html blog post file into the repo folder.
  2. Goto GitHub click the "blogs" folder
  3. Click upload file and select the html file in your downloads
  4. Next on the left hand side you'll see index.html click on that
  5. Click the pencil in the top left side of the editor
  6. Scroll down until you see something like "a class ='blog-entry'"
  7. Copy and paste the template that is already there with explanations
  8. Now change the href="blogs/NAME HERE.html" to the name of the html file you just added
  9. Scroll up and click commit changes!

Now your blog post is live on GitHub Pages!
